ShareFramework
==============

Description
ShareFramework helps you to share content with others apps in iOS

Frameworks
 - MessageUI.framework
 - Social.framework
 - Accounts.framework


Add to your "Supporting Files/{name}-Prefix.pch" 
 - #import "ShareHeaders.h"
